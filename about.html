About

This website is powered by NASA's Open APIs. More info about these APIs can be found at https://api.nasa.gov. These APIs are very powerful and free to use for the public, allowing access to lots of information in NASA's databases.

I created this website in 2023 as part of my Computer Science 3 class, and as an introduction into APIs and into web design.
